# Global Piano Movers

A premium, conversion-focused one-page website for **Global Piano Movers** — specialist piano moving, transportation, storage and international shipping across the **United States, United Kingdom & Canada**.

## Overview

- **Single self-contained page** — [`index.html`](index.html) with embedded CSS & JS, no build step required. Just open the file or serve the folder.
- **Design language** — luxury logistics editorial aesthetic: floating glass panels, warm gold glow, monospace technical labels, mixed uppercase + italic-serif headlines.
- **Palette** — Deep Navy, Piano Black, Gold accents on white/warm-paper backgrounds.
- **3D & motion** — perspective tilt on the hero and cards, scroll-reveal animations, animated stat counters, testimonial carousel, FAQ accordion.
- **Conversion-focused** — multiple quote CTAs, click-to-call links, and a quote-request form.
- **SEO** — title/meta/keywords, Open Graph + Twitter cards, and JSON-LD schema (MovingCompany, Service catalog, FAQPage).

## Sections

Hero · Why Pianos Require Specialized Movers · Services · Process · What's Included · Coverage (US/CA/UK) · Why Choose Us · Stats · Testimonials · FAQ · Emergency Moving · Final CTA · Contact · Footer.

## Running locally

Open `index.html` directly, or serve the folder:

```bash
python -m http.server 5500
# then visit http://localhost:5500/index.html
```
